localhost မှသည် yourname[dot]com စီသို့

ပြီးခဲ့တဲ့ တစ်ခေါက်ကတော့ ကိုယ့်ရဲ့ HTML,CSS,JS,PHP,MySQL နဲ့ ရေးထားတဲ့ Web Project တစ်ခုကို free shared hosting တစ်ခုမှာ ဘယ်လိုတင်ကြလဲဆိုတာ မျှဝေခဲ့ပါတယ်။ ဒီတစ်ခေါက်ကတော့ ကိုယ့်ရဲ့ web project တစ်ခုကို VPS တစ်ခုပေါ် ဘယ်လို တင်ကြလဲ ဆိုတာကို မျှဝေပေးချင်ပါတယ်။

အလုပ်ကတော့ shared hosting မှာ တင်ရတာထက် ပိုရှုပ်ပါတယ်။ shared hosting မှာက file upload တင်တာတွေ၊ database mangement လုပ်တာတွေကို သူတို့ပေးထားတဲ့ web panel မှာတင် အလွယ်တကူလုပ်လို့ ရပေမယ့် အခု VPS တွေမှာတော့ သူတို့ပေးထားတာက linux terminal တစ်ခုပဲ ရှိပါတယ်။ ဒီတော့ linux command တွေနဲ့ ပတ်သက်ပြီး သေစာရှင်စာလောက်တော့ ရေးတက် ဖတ်တက်မှ VPS သုံးရတာ အဆင်ပြေပါလိမ့်မယ်။

VPS တွေကို ရောင်းတဲ့ ကုမ္မဏီ အမျိုးမျိုး စျေးအမျိုးမျိုး ရှိတဲ့အထဲက USD 100 အလကား ရတဲ့ Digital Ocean ကို သုံးပြီး မျှဝေသွားပါမယ်။
လုပ်ရမယ့် အလုပ်တွေက

၁။ Digital Ocean မှာ account register လုပ်ပြီး New Droplet ခလုပ်ကို နှိပ်ပါ။ ကိုယ်လိုချင်တဲ့ specification ( ram, processor, storage စတာတွေ  ) ရွေးပြီးပါ။

၂။ အသစ် ရလာတဲ့ droplet စတင်အလုပ်လုပ်ဖို့ တစ်မိနစ်လောက် စောင့်ရပါတယ်။

၃။ droplet မှာ ပါတဲ့ ip address ကို ဖွင့်ကြည့်ရင် apache welcome page ကို မြင်ရမှာ ဖြစ်ပါတယ်။
