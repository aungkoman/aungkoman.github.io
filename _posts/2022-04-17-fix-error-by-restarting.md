### Flutter မှာ Error ရှင်းနည်း
Error တစ်ခုတက်တဲ့အခါ ... 
Hot Reload ကို မျက်စိမှတ် ယုံမနေပဲ ကျွန်ပျူတာကို restart ချပြီးမှ run ကြည့်ပါ။

ဒါကတော့ အရှင်းဆုံး ပြောတာပါ။
hot reload ကို အကုန်ယုံပြီး ပုံလိုက်ရင် ငါးပါးမှောက်ပါလိမ့်မယ်။
ဒါကြောင့် major changes တွေ လုပ်ပြီးတိုင်းမှာ app ကို အစအဆုံး ပြန် build ပါ။
တစ်ချို့ error တွေက hot reload ကြောင့် တက်တာမျိူး။
ဉပမာ Package အသစ်တွေ သွင်းတဲ့အခါ တန်းမသိတာမျိုး။
const တွေ တန်ဖိုး ပြင်တဲ့အခါ အလုပ်မလုပ်တာမျိုး။

UI ပြင်တာအတွက်ကတော့ hot reload အတော်ကို မိုက်ပါတယ်။
မြန်ပါတယ်ဆိုတဲ့ သာမန် HTML, CSS, JS web development မှာတောင် ဒီလောက် မမြန်။
